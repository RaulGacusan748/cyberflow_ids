1. The Telemetry Landscape

The core operational challenge is the severe imbalance of network data. Malicious activity is a needle in a haystack. CyberFlow processes the UNB CIC-IDS2017 dataset using a 5% Stratified Downsampling Strategy to protect minority attack signatures.

Class Distribution (126,038 Flows)

Standard classifiers default to the majority class to minimize raw error rates. We optimize for F1-Score to counteract this.

📊 The Packet Asymmetry Metric

Volumetric floods exhibit highly asymmetrical profiles (spraying packets with no replies). We engineered a dimensionless metric to isolate this footprint.

Asymmetry = (Fwd_Pkts - Bwd_Pkts) / (Fwd_Pkts + Bwd_Pkts + 10⁻⁵)

2. The Machine Learning Tournament

A four-way tournament evaluated distinct mathematical classifiers. To protect the SOC from alert fatigue, Precision and F1-Score were the primary KPIs.

Classifier Empirical Results

Evaluated on a 20% validation split (25,208 records).

Champion: XGBoost
Latency: 2.69s (Train) | <25ms (Inference)

👑 Why XGBoost Won

Secured a peak F1-Score of 98.68% and Precision of 99.10%. Its sequential gradient boosting architecture minimizes error margins by continuously targeting misclassified residual weights.

📉 Why Naive Bayes Failed

Yielded an unusable 31.81% precision. In network traffic, features (packet size, duration) are highly correlated. Naive Bayes' assumption of absolute independence was severely violated.
