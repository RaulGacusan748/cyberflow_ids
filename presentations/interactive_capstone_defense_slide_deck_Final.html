The Challenge of Modern Networks
Security Context
  • Volumetric Assaults: Modern high-velocity vectors like DDoS floods saturate hardware interfaces before signature firewalls can adapt.
  • Severe Class Imbalance: Malicious records routinely constitute under 17% of overall telemetry, causing baseline classifiers to ignore intrusions.
  • Alert Fatigue: Standard IDS systems generate thousands of complex, unexplained alarms, leading to analyst burnout and delayed response times.

Project Goal

To design and implement a memory-safe, cost-sensitive classification pipeline that detects threats in real-time (<25ms) and leverages Autonomous Agent Orchestration to explain and mitigate incidents instantaneously.

Engineering The Packet Asymmetry Metric
Mathematics

Volumetric attacks and network sweeps exhibit severe directional imbalances. We mathematically isolated this behavioral feature by engineering a custom dimensionless imbalance ratio.

Packet Asymmetry = Packetsfwd - Packetsbwd Packetsfwd + Packetsbwd + ε

Where ε = 10^-5 serves as a numeric guardrail preventing division-by-zero errors on dead connection lines.
